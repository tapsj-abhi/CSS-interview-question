Q.2 Explain inline , internal and external stylesheet

Inline Stylesheet: CSS styles applied directly to an HTML element using the style attribute within the tag.
Internal Stylesheet: CSS styles defined within a style tag in the head section of an HTML document.
External Stylesheet: CSS styles defined in a separate .css file and linked to the HTML document using the link tag.
